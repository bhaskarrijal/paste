== Description ==
Allows users to paste images directly into the WordPress media library upload modal via Ctrl+V (Windows/Linux) or Cmd+V (Mac).

== Installation ==
1. Upload the `paste-image-uploader` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the Media Library and open the upload modal.
4. Press `Ctrl+V` (or `Cmd+V` on Mac) to paste an image from your clipboard.
5. The image will be uploaded and attached to your media library.

== Frequently Asked Questions ==
= What file types are supported? =
The plugin supports all image types supported by WordPress (PNG, JPEG, GIF, BMP, etc.).

= Can I disable console logging? =
Currently, minimal logging is done via the browser console. Future updates may include settings to toggle logging.
