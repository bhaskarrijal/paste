Extremely lightweight plugin to paste images directly from clipboard into the WordPress media library upload modal.

== Description ==
Paste Image Uploader is a simple yet powerful tool that allows users to paste images directly into the WordPress media library upload modal via Ctrl+V (Windows/Linux) or Cmd+V (Mac).

**Key Features:**

* Paste images directly from clipboard into any WordPress media upload modal
* Simple settings page to enable/disable the plugin functionality
* Native WordPress auto-update support
* Extremely lightweight with minimal impact on your site's performance
* Quick access to settings through plugin listing page

The plugin is perfect for content creators who frequently work with screenshots or images from other applications and want to streamline their workflow.

== Installation ==
1. Upload the `paste-image-uploader` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Paste to configure the plugin (optional).
4. Go to the Media Library and open the upload modal.
5. Press `Ctrl+V` (or `Cmd+V` on Mac) to paste an image from your clipboard.
6. The image will be uploaded and attached to your media library.

== Frequently Asked Questions ==
= What file types are supported? =
The plugin supports all image types supported by WordPress (PNG, JPEG, GIF, BMP, etc.).

= How do I access the plugin settings? =
Navigate to Settings > Paste in your WordPress dashboard, or click the "Settings" link on the plugins page.

= Can I disable the plugin temporarily? =
Yes, you can disable the paste functionality from the plugin settings without deactivating the entire plugin.

= Does the plugin support auto-updates? =
Yes, the plugin supports WordPress native auto-updates by default.
